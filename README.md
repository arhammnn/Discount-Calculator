# Discount Calculator 🧮

Sebuah aplikasi web sederhana untuk menghitung total harga setelah dipotong diskon. Proyek ini dibangun menggunakan bahasa pemrograman Python dengan *framework* Flask untuk mengelola sisi *backend* dan *routing*.

# Fitur Utama
* Menghitung total harga kotor berdasarkan harga satuan dan kuantitas barang.
* Mengkalkulasi otomatis nominal potongan harga berdasarkan persentase diskon.
* Menampilkan total harga bersih (Total Bayar) yang harus dibayarkan.
* Antarmuka pengguna (UI) yang responsif dan mudah dipahami.

# Teknologi yang Digunakan
* **Backend:** Python, Flask
* **Frontend:** HTML5, CSS3

# Cara Menjalankan Aplikasi di Komputer Lokal

Pastikan Python sudah terinstal di komputer Anda sebelum menjalankan aplikasi ini.

1. Buka Terminal atau Command Prompt.
2. *Clone repository* ini ke dalam komputer Anda:
   ```bash
   git clone [https://github.com/arhammnn/Discount-Calculator.git](https://github.com/arhammnn/Discount-Calculator.git)

   Masuk ke dalam folder proyek:
   cd Discount-Calculator

   Instal library Flask (jika sebelumnya belum terinstal):
   pip install flask

   python calculator.py

   Buka browser dan kunjungi alamat URL lokal yang muncul di terminal http://127.0.0.1:5000/
